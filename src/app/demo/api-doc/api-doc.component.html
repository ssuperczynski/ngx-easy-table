API Documentation

Table has functionality to control it's content outside component. It means by using API you can control table's behaviour from the parent component.

To define API object use code snippet defined below:

Type Code Returns Description
API.setInputValue
void Set input value. You can set array of values.
API.rowContextMenuClicked
number Emit event to close context menu
API.toggleRowIndex
number Emit event to toggle additional row details
API.toggleCheckbox
number Emit event to toggle row checkbox
API.onGlobalSearch
number Emit event to filter table by all the columns
API.setRowClass
void Set table row class
API.setCellClass
void Set table cell class
API.setRowStyle
void Set table row style
API.setCellStyle
void Set table cell style
API.setTableClass
void Set table class (for example Bootstrap)
API.setPaginationCurrentPage
void Set pagination on given page
API.getPaginationTotalItems
number Get pagination total items
API.getNumberOfRowsPerPage
number Get current page items amount
API.getPaginationCurrentPage
number Get pagination current page
API.sortBy
void Set sort by column
API.setPaginationDisplayLimit
void Set amount of row displayed.
